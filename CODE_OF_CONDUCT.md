# Code of Conduct

## Our Standard

This project expects respectful, technically grounded collaboration.

Participants should:

- communicate clearly and professionally
- focus on reproducible facts, operator impact, and concrete evidence
- welcome critique of ideas without making it personal
- keep discussions usable for operators, maintainers, contributors, and potential supporters

Participants must not:

- harass, insult, or intimidate others
- post private or sensitive information without permission
- derail technical discussions with hostility, spam, or bad-faith behavior
- use issues or pull requests as a vehicle for abuse or commercial pressure

## Scope

This code of conduct applies to repository discussions, issues, pull requests, review comments, and other project spaces controlled by the maintainer.

## Enforcement

The maintainer may edit, hide, lock, or remove comments, issues, pull requests, or participants that violate these expectations.

## Reporting

If you experience or observe unacceptable behavior, use the maintainer contact path described in [SUPPORT.md](SUPPORT.md) or [SECURITY.md](SECURITY.md), depending on the sensitivity of the situation.
